# テプリッツ行列パーマネント最小値探索

## 目的
テプリッツ行列に特化して、Kräuter予想の最小正パーマネント値 `2^{n - ⌊log₂(n + 1)⌋}` を達成する行列を探索する。

## やりたいこと
1. 特定のサイズnの(±1)テプリッツ行列について考える
2. すべてのパターンのテプリッツ行列を生成し、パーマネントを計算
3. 正の整数パーマネント値の中で最小値を特定
4. 予想値 `2^{n - ⌊log₂(n + 1)⌋}` と一致する行列があれば出力

## Kräuter予想 (Conjecture 36)
> The minimum positive value taken by P on the set of (+1, -1)-matrices of order n is  
> **2^{n - ⌊log₂(n + 1)⌋}**

### 予想値の例
- n=2: 2^{2-1} = 2
- n=3: 2^{3-2} = 2  
- n=4: 2^{4-2} = 4
- n=5: 2^{5-2} = 8

## UX フロー
1. Pythonプログラムを実行
2. 行列サイズ n を入力
3. すべてのテプリッツ行列パターンでパーマネント計算
4. 予想値と一致する行列を発見したら値と行列を出力
5. 最小正パーマネント値の分析結果を表示

## テプリッツ行列の定義
集合 S に対して、T_{n,S} を次のように定義：
- (i,j) 成分は j-i ∈ S なら -1、そうでなければ +1

例)
T_8{2..4,6}
T_9{2,3,5..7}
T_10{-5..1,5..7}


### 生成戦略
1. **全集合パターン**: S ⊆ {-(n-1), ..., 0, ..., (n-1)} のすべての組み合わせ
2. **対称パターン**: S = -S の対称な集合のみ
3. **連続パターン**: S = {a, a+1, ..., b} の連続区間
4. **スパースパターン**: |S| が小さい場合

## 計算量見積もり
- テプリッツ行列数: 2^{2n-1} (集合Sの選び方)
- 各パーマネント計算: O(2^n × n) (Ryser公式)
- 総計算量: O(2^{3n-1} × n)
